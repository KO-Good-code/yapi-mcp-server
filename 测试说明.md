# 测试说明

## 当前问题

YApi 是一个单页应用（SPA），接口数据通过 JavaScript 动态加载，不在 HTML 源码中。

## 解决方案

### 方案1: 使用 YApi API（推荐）

需要提供有效的 Token：

```bash
# 设置环境变量
export YAPI_BASE_URL="https://yapi.makeblock.com"
export YAPI_TOKEN="your-real-token"

# 测试
./test.sh 'https://yapi.makeblock.com/project/382/interface/api/17888'
```

### 如何获取 YApi Token

1. 登录 YApi: https://yapi.makeblock.com
2. 进入项目设置
3. 找到"Token配置"
4. 复制 Token

### 方案2: 直接调用 YApi API

测试 API 是否可访问：

```bash
# 不需要 token 的测试
curl "https://yapi.makeblock.com/api/interface/get?id=17888"

# 需要 token 的测试
curl -H "Authorization: Bearer your-token" \
     "https://yapi.makeblock.com/api/interface/get?id=17888"
```
