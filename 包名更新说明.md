# 包名更新说明

## 更新内容

包名已从 `yapi-mcp-server` 更新为 `@neigri/yapi-mcp-server`

## 原因

- `yapi-mcp-server` 已被其他用户占用
- 使用 scope 包名 `@neigri/yapi-mcp-server` 可以避免冲突

## 使用方式

### 安装

```bash
# 全局安装
npm install -g @neigri/yapi-mcp-server

# 或使用 npx（推荐）
npx -y @neigri/yapi-mcp-server@latest
```

### 在 Claude Desktop 中配置

```json
{
  "mcpServers": {
    "yapi-mcp-server": {
      "command": "npx",
      "args": [
        "-y",
        "@neigri/yapi-mcp-server@latest"
      ],
      "env": {
        "YAPI_BASE_URL": "https://yapi.makeblock.com",
        "YAPI_TOKEN": "your-token"
      }
    }
  }
}
```

### 在 Cursor 中配置

```json
{
  "mcpServers": {
    "yapi-mcp-server": {
      "command": "npx",
      "args": [
        "-y",
        "@neigri/yapi-mcp-server@latest"
      ],
      "env": {
        "YAPI_BASE_URL": "https://yapi.makeblock.com",
        "YAPI_TOKEN": "your-token"
      }
    }
  }
}
```
